class log.Log(parent, basicData={})¶

基类：PySide2.QtWidgets.QWidget

日志类，负责输出到日志文件和显示。

InitLog()¶

初始化日志格式

createLayout()¶

初始化日志视图

print(logStr, level=20, popup=False, buttons=1, modal=True)¶

打印日志在日志窗口，并控制父级状态栏显示，同时写入到日志文件。

>>> print("测试输出内容",20,True)
True
参数

  • logStr (str) – 需要打印的内容。

  • level (int) – 打印日志的等级：CRITICAL = 50,ERROR = 40,WARN = 30,INFO = 20,DEBUG = 10,NOTSET = 0

  • popup（bool） – 是否弹窗显示。

  • buttons（bool） – 弹出窗口的按钮数量，还可以是2，弹出窗口显示yes或no，返回bool值。

  • modal（bool） – 是否模态显示窗口。

返回

返回是否执行成功，有弹窗返回选择结果。

返回类型

bool
