API¶

RPA.RPA(parent)¶
RPA包含以下模块：

RPA.log = Log(self, basicData)； RPA.parameter = Parameter(self,parameterList)； RPA.command = Command(self,commandList)； RPA.table = Table(self,viewList)； RPA.browser = Browser(self, controlDict)

已导入以下库：

json，os，sys，time，pandas as pd

其他库可以自行在脚本中导入，要注意库文件路径是否可以访问到。

RPA.Log.print(self, logStr, level=20, popup=False, buttons=1, modal=True)¶

打印日志在日志窗口，并控制父级状态栏显示，同时写入到日志文件。

>>> print("测试输出内容",20,True)
True
参数

  • logStr (str) – 需要打印的内容。

  • level (int) – 打印日志的等级：CRITICAL = 50,ERROR = 40,WARN = 30,INFO = 20,DEBUG = 10,NOTSET = 0

  • popup（bool） – 是否弹窗显示。

  • buttons（bool） – 弹出窗口的按钮数量，还可以是2，弹出窗口显示yes或no，返回bool值。

  • modal（bool） – 是否模态显示窗口。

返回

返回是否执行成功，有弹窗返回选择结果。

返回类型

bool

RPA.Browser.browser(self, oper, value='')¶

使用浏览器自动化执行一些操作

>>> browser("打开网址", "https://www.baidu.com/")
True
>>> browser("截图", "c:\test.jpg")
True
参数

  • oper (str) – 操作方法：[“打开网址”, “刷新”, “上一页”, “下一页”, “获取网址”, “获取标题”, “获取代码”, “执行JS”, “下拉加载”, “弹窗自动同意”, “弹窗自动拒绝”, “伪装浏览器”, “设置代理”, “截图”, “另存为PDF”, “下载”]

  • value (str) – 一些操作需要传入的参数，如打开网址的网址。

返回

返回是否执行成功，或者执行后获取的值。

返回类型

bool, str

RPA.Browser.browserExp(self, oper, control={}, contIndex='', value='')¶

使用浏览器，操作控件，执行一些操作。

>>> browserExp("输入", "#q", 0, "测试输入")
True
>>> browserExp("点击", "搜索按钮")
True
参数

  • oper (str) – 需要执行的操作：[“点击”, “模拟点击”, “获取值”, “获取代码”, “获取属性”, “输入”, “模拟输入”, “上传文件”, “计数”, “右击”, “获取标签”, “获取坐标”, “获取尺寸”, “截图”, “下拉加载”, “是否显示”, “等待加载”]

  • control (str) – 可以是提前设置好的控件名称，也可以直接是一个css选择器字符串。

  • contIndex（int） – css选择器选择一组控件时，需要指定操作的控件索引。

  • value（str） – 一些操作需要传入的参数，如输入。

返回

返回是否执行成功，或者执行后获取的值。

返回类型

bool, str
