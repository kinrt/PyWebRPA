2. browser module¶

class browser.Browser(parent, controlDict={})¶

基类：PySide2.QtWidgets.QWidget

browser(oper, value='')¶

使用浏览器自动化执行一些操作

>>> browser("打开网址", "https://www.baidu.com/")
True
>>> browser("截图", "c:\test.jpg")
True
参数

  • oper (str) – 操作方法：[“打开网址”, “刷新”, “上一页”, “下一页”, “获取网址”, “获取标题”, “获取代码”, “执行JS”, “下拉加载”, “弹窗自动同意”, “弹窗自动拒绝”, “伪装浏览器”, “设置代理”, “截图”, “另存为PDF”, “下载”]

  • value (str) – 一些操作需要传入的参数，如打开网址的网址。

返回

返回是否执行成功，或者执行后获取的值。

返回类型

bool, str

browserExp(oper, control={}, contIndex='', value='')¶

使用浏览器，操作控件，执行一些操作。

>>> browserExp("输入", "#q", 0, "测试输入")
True
>>> browserExp("点击", "搜索按钮")
True
参数

  • oper (str) – 需要执行的操作：[“点击”, “模拟点击”, “获取值”, “获取代码”, “获取属性”, “输入”, “模拟输入”, “上传文件”, “计数”, “右击”, “获取标签”, “获取坐标”, “获取尺寸”, “截图”, “下拉加载”, “是否显示”, “等待加载”]

  • control (str) – 可以是提前设置好的控件名称，也可以直接是一个css选择器字符串。

  • contIndex（int） – css选择器选择一组控件时，需要指定操作的控件索引。

  • value（str） – 一些操作需要传入的参数，如输入。

返回

返回是否执行成功，或者执行后获取的值。

返回类型

bool, str

getElement(control, contIndex=0)¶

选择浏览器的控件，默认按照 CSS选择器 传入控件路径 新增功能：

  • && 适用于控件在部分DOM中没有的情况，自动拆分为多个路径，分多次获取最终路径。

  • :nth-child(0) 增加子元素选择器参数为0的选项，这样可以选择所有子元素。

参数

  • control (str) – 控件名称，或者css选择器字符串

  • contIndex (int) – 需要操作的控件索引编号

返回

返回控件的一些参数，如：”offsetTop”，”offsetLeft”，”scrollTop”，”scrollLeft”

返回类型

dict
